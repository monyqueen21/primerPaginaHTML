HTML: es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la sigla que corresponde a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto, que podría ser traducido como Lenguaje de Formato de Documentos para Hipertexto. Versiones de HTML:
  • HTML 2.0
  • HTML 3.2
  • HTML 4.01
  • HTML 5.0
